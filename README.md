# MyVibe - Découvreur d'Expériences à Yaoundé

MyVibe est une plateforme web qui recommande des films, livres et lieux à Yaoundé selon les goûts de l'utilisateur.

## Équipe

- Membre 1 : UI/UX & Intégration (HTML/CSS)
- Membre 2 : Frontend JavaScript(INTERACTIVITE)
- Membre 3 : Backend PHP(LE CERVEAU)
- Membre 4 : Données (JSON / lieux / livres / films, SQL SPECIALISTE)
- Membre 5 : QA & DevOps (QUALITE ET CONTROLE)

## Technologies

- HTML5
- CSS3
- JavaScript Vanilla
- PHP
- JSON
- Git & GitHub
